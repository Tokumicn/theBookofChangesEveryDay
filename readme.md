

功能简述：

主要功能：
1. 随机产生一卦，并给予描述；
2. 查询卦象的具体描述；
3. 每一爻的含义，爻变的结果等；


辅助功能：
1. 爬虫获取卦象信息；
2. 数据存储在本地文件；
3. 在控制台画出对应的卦象；


卦数据结构抽象：
1. 6位0-1字符串；
2. 0代表阴爻，1代表阳爻；
3. 索引0为上爻，索引5为初爻；
4. 例如："000000"：坤为地卦  "111111"：乾为天卦
